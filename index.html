Generador de CV aleatorios

¿Qué hace esta página?

Esta página, como indica su título, es un generador de currículums con contenido aleatorio. Cada vez que se clickea en la barra del navegador "Generar Nuevo CV aleatorio" un nuevo CV de una persona ficticia es generado. Se puede utilizar tantas veces como sea necesario. Los datos personales son tomados de la página "RANDOM USER GENERATOR", mientras que para la secciones "Experiencia" y "Habilidades" se habilita un formulario para cada una de ellas, mediante el cual puede completarse esa información manualmente. Para finalizar, es posible ocultar cada uno de los formularios (y volverlos a mostrar).

Espero lo disfruten.
